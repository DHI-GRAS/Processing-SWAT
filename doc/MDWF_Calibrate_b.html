Generate PEST instruction file

Instruction files tell PEST where to find simulated observations. At the moment, only river discharge observations are supported by the module.

Select model source folder

Folder where instruction file will be saved to.

File with observation data (time, obs, error)

River discharge observations need to be provided as simple .csv files with the date in the first column, the discharge (m3/s) in the second column and the error of the observation in the third column. Dates should be EXCEL dates converted to numbers.

PEST name of observation group

Name of obervation group which the observed data should be long to.

Subbasin corresponding to observations

Subbasin ID corresponding to observations.

Number of subbasins in the model

The total number of subbasins in the model.
